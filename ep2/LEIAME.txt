USO:

Para compilar:

	Não é necessário compilar


Para executar o ep2.py no terminal:
 	 
 	- python ep2.py

_______________________

OBSERVAÇÃO:

	- Eh necessaria instalacao previa da linguagem Python 2.7


_______________________

COMO EXECUTAR:

Após executar 'python ep2.py' no terminal, você deve inserir os parametros r q g k e k sequencias que serão alinhadas S1 S2 S3 ... Sk

É necessário usar espaços entre os parâmetros e não virgula, ponto ou algo do tipo. O ep não faz tratamento de erros nos parâmetros.

Uma ilustração de como executar está se encontra junto com os arquivos.
